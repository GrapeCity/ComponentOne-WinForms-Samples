Style Sheets

We can handle style sheets. This page has a green Tahoma body with 1-cm margins all around. The blue bold face was applied to the paragraph style (<p>).

There's also a class style (this one, large Gothic red on bisque, it's called 'MyClass') and an element style (this one, large Green on Yellow, it's called 'MyElement'). These styles are applied to spans.

This paragraph also has style "MyClass". In addition to changing fonts and colors, it also centers paragraphs.
