## HtmlGrid
#### [Download as zip](https://minhaskamal.github.io/DownGit/#/home?url=https://github.com/GrapeCity/ComponentOne-WinForms-Samples/tree/master/NetFramework\FlexGrid\VB\HtmlGrid)
____
#### Shows how to display and edit Html text in grid cells.
____
The sample uses a C1SuperLabel control to render Html text in grid cells, and a C1Editor control to edit the Html content.
