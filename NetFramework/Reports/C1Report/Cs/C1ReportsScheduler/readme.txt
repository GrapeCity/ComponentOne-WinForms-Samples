C1ReportsScheduler
------------------------------------------------------------------------
C1ReportsScheduler frontend application and Windows service.

Provides a Windows service C1ReportsSchedulerService that can be used to
run (generate and export or print) C1Report and C1PrintDocument objects
on schedule at specified times and intervals. For details, run the
C1ReportsScheduler.exe frontend, and click Help | Contents.

That sample requires .NET 4.0/Visual Studio 2010 or later.
