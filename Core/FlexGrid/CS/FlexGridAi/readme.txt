GeminiAI_FlexGrid
-----------------------------------------------------------------------------
FlexGridAI is a demo application that showcases the integration of the C1 FlexGrid WinForms control with Gemini AI. 
It demonstrates how users can generate dynamic, structured data using natural language prompts and visualize the output within a FlexGrid.

Key Features
	Users input natural language prompts
	The Gemini AI model processes the request
	The AI response is transformed into structured tabular data and displayed in a FlexGrid

Supported Prompt Categories
	Comparisons
	Aggregation
	Extremes

Important Notes
	API Key Handling: The application requests the user to enter their Gemini API key.
	Environment Variable Storage: The entered API key is stored as an environment variable named GEMINI_API_KEY on the user's system.
