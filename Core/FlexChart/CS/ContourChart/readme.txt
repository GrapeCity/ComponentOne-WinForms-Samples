ContourChart
--------------------------------------------
ContourChart is a sample application that demonstrates the usage of the C1 FlexChart Extended Contour Chart in WinForms.
It shows how 2D numerical data can be visualized using contour lines and color-filled regions to reveal patterns, gradients, and value boundaries.

Key Features
	Visualizes 2D array-based data on a contour surface
	Supports both automatic and manually defined contour levels
	Uses gradient color scales for intuitive heat-style visualization
	Integrates seamlessly with FlexChart as a chart series

Supported Contour Properties
	Appearance
		LineArea
		Area
		Line

	NumberOfLevels – Automatically divides the data range into levels
	Levels – Allows specifying exact contour values
	ColorScale – Controls min/max values and gradient colors
	DataSource – Accepts 2D numeric arrays
