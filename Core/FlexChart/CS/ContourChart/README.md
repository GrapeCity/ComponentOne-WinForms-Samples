## ContourChart
____
#### Demonstrates the usage of the C1 FlexChart Extended Contour Chart in WinForms.
____
The sample shows how to:

* Visualize 2D array-based numerical data on a contour surface.
* Use automatic or manually defined contour levels.
* Apply gradient color scales for heatmap-style visualization.
* Integrate the contour chart as a series within a FlexChart.

Supported Contour Properties:

* **Appearance**
  * LineArea
  * Area
  * Line

* **NumberOfLevels** – Automatically divides the data range into levels.
* **Levels** – Specify exact contour values manually.
* **ColorScale** – Control min/max values and gradient colors.
* **DataSource** – Accepts 2D numeric arrays.
