InputDateRange
-----------------------------
Interactive Date Range Picker

This sample demonstrates a comprehensive date range picker application built with ComponentOne controls C1CalendarView. The application provides an interface for selecting date ranges through multiple methods including predefined options.

Key features include:
- Multi-cultural support with localized UI text for English (en-US), Korean (ko-KR), and Japanese (ja-JP)
- Dual-month calendar display with custom visual highlighting for selected date ranges
- Predefined date range options with automatic matching and synchronization
- Configurable date limits (min/max dates) and selection count restrictions
- Culture-specific date formatting and ordinal number display
- Apply/Cancel pattern for confirmed vs temporary selections
