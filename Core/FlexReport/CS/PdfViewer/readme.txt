PdfViewer
----------------------------------------
Demonstrates to use C1FlexViewer to view Pdf documents

This sample shows how the C1PdfDocumentSource component can be used
with C1FlexViewer to view PDF document.

The component open PDF document and shows in C1FlexViewer.
Set the C1PdfDocumentSource.DocumentLocation property to PDF document's path
to open it in the linked C1FlexViewer viewer instance.
