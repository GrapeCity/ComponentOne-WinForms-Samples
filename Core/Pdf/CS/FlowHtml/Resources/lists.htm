Lists

Html has two main types of lists: unordered (<UL>) and ordered (<OL>). Here's an unordered list:

  • Apples
  • Oranges
  • Bananas

And here's an ordered list:

  1. Apples
  2. Oranges
  3. Bananas

And here's a tricky list that mixes regular paragraphs with list items:

    This looks like an indented paragraph, but it's just plain text in a list.

    And this is a paragraph in a list.

That's nice and plain. Let's spice it up with some styles and line breaks:

  • Apples:
    A very nice fruit, red or green, from Washington or elsewhere. Remember, one per day keeps the doctor away. My favorites are (we're nesting lists now):
    1. Fuji,
    2. Gala, and
    3. Granny
  • Oranges:
    As nice as apples, but messier and harder to peel.
  • Bananas:
    My favorite. Perfect packaging, sweet and nutricious.

Unordered lists come in these styles:

type = "circle"
  • ready
  • aim
  • fire
type = "square"
  • ready
  • aim
  • fire
type = "disc"
  • ready
  • aim
  • fire

Ordered lists come in these styles:

type = "1"
  1. ready
  2. aim
  3. fire
type = "a"
  1. ready
  2. aim
  3. fire
type = "A"
  1. ready
  2. aim
  3. fire
type = "i"
  1. ready
  2. aim
  3. fire
type = "I"
  1. ready
  2. aim
  3. fire
