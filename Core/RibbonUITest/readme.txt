UIAutomationDemo
--------------------------------------------
This UI Automation Test Suite is designed to validate the functionality and UI Accessibility of the C1Ribbon control within a WinForms WordPad application. The suite leverages advanced UI testing techniques to ensure comprehensive control interaction.

Key Technologies Used:

Automation Framework: Appium WebDriver
Test Framework: xUnit
Target Application: WinForms WordPad with C1Ribbon Control

System Requirements:
.NET 8.0 SDK
Visual Studio 2022 or later
Node.js v22.14.0 (for Appium)
Appium CLI v2.15.0
WinAppDriver v1.3

Refer the USERMANUAL.md file in the project for setup guide.
